==============================================================================
 1. INTRODUCTION                                                  *lotr-intro*

LOTR -- Lord of the Regs -- A persistent view of :registers in a sidebar window.

==============================================================================
 2. CONFIGURATION                                         *lotr-configuration*

These options can be set in your |$MYVIMRC| to modify the behaviour of LOTR:

|'lotr_position'|	Position the LOTR window (left, right, top, bottom)
|'lotr_winsize'|	Width or height of the LOTR window
|'lotr_expand'|		Force GVim to widen if necessary to accommodate LOTR
|'lotr_yankstack'|	Show the :Yanks list from the YankStack plugin

------------------------------------------------------------------------------
                                                             *'lotr_position'*
Values: string (left, right, top, bottom)                                    ~
Default: right                                                               ~

------------------------------------------------------------------------------
                                                              *'lotr_winsize'*
Values: number                                                               ~
Default: 25                                                                  ~

------------------------------------------------------------------------------
                                                               *'lotr_expand'*
Values: boolean                                                              ~
Default: 0                                                                   ~

By default the LOTR window will not cause GVim to resize to accommodate it.

------------------------------------------------------------------------------
                                                        *'lotr_focus_on_open'*
Values: boolean                                                              ~
Default: 0                                                                   ~

By default the LOTR window will not take focus when opened.

------------------------------------------------------------------------------
                                                             *'lotr_map_keys'*
Values: boolean                                                              ~
Default: 1                                                                   ~

By default LOTR will add key maps.

------------------------------------------------------------------------------
                                                            *'lotr_yankstack'*
Values: boolean                                                              ~
Default: 1                                                                   ~

If installed, LOTR will show the YankStack list as shown by the :Yanks command.

==============================================================================
 3. COMMANDS                                                   *lotr-commands*

The following commands are available:

`:LOTROpen`	Open the LOTR window

`:LOTRClose`	Close the LOTR window

`:LOTRToggle`	Toggle the open/close state of the LOTR window


The `:LOTRToggle` command is also available through `<PLUG>LOTRToggle`
which is mapped to `<leader>cr` by default. If `g:lotr_map_keys` is set
to 0 then the map will not be added. You can change this by adding a line
like this to your |$MYVIMRC|:
>
  nmap <leader>r <plug>LOTRToggle
<


==============================================================================
 4. LOTR WINDOW COMMANDS                                *lotr-window-commands*

From within the LOTR window:

`<Enter>`	Jumps back to the prior window (|wincmd| p)
`<Space>`	Toggles zoom mode
`q`		Closes the LOTR window
